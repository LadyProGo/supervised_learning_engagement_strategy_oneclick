Проект по удержанию клиентов в интернет-магазине «В один клик».

Описание проекта:

Интернет-магазин «В один клик» продаёт товары для детей, дома, косметику, технику и продукты. Отчёты показали снижение активности постоянных клиентов, при этом привлекать новых стало неэффективно. Цель проекта — удержание постоянных клиентов с помощью персонализированных предложений на основе анализа данных.

Постановка задачи:

- Построить модель, предсказывающую вероятность снижения активности клиентов.
- Выделить наиболее прибыльных клиентов в зоне риска.
- Разработать рекомендации для удержания этих клиентов.

Были протестированы 4 модели:

- DecisionTreeClassifier
- KNeighborsClassifier
- LogisticRegression (с L1-регуляризацией)
- SVC (с различными ядрами и параметрами C)

Подбор параметров осуществлялся через RandomizedSearchCV с кросс-валидацией и метрикой ROC-AUC.

Лучшая модель:

Support Vector Classifier (SVC) с параметрами:

- C = 10
- kernel = 'rbf'
- probability = True

Качество модели:

- ROC-AUC на кросс-валидации: 0.901
- ROC-AUC на тестовой выборке: 0.908

Выводы:

Был выделен сегмент клиентов с вероятностью снижения активности более 0.7 и высокой прибыльностью (входит в топ-25%).

Характеристики сегмента:

- Меньше времени на сайте и меньше просмотров
- Более высокая медианная выручка и прибыль
- Чувствительность к скидкам
- Давние клиенты, возможно, испытывают усталость от ассортимента

Рекомендации по удержанию:

- Персонализированные скидки и предложения на основе истории покупок.
- Уведомления о брошенных корзинах, снижении цен, наличии товара.
- Упрощение интерфейса: быстрый доступ к любимым товарам и акциям.
- Механики вовлечения: бонусы за активность, ранний доступ к новинкам.
- VIP-сервис: персональные предложения, ускоренная доставка, доступ к закрытым распродажам.

Заключение:

Сегмент небольшой, но очень прибыльный. Потеря таких клиентов ощутимо скажется на бизнесе. Им не нужна массовая реклама — важны индивидуальный подход и удобство.
